Final Year Project

The Relationship between bullet velocity and penetration:

For my final year project artefact and dissertation I am exploring projectile motion and how bullet's behave when impacting a target. Many video games include object penetration through weaker medias such as a wooden board, but do not accurately equate the relevant damage output through the change in velocity. By exploring different penetration methods such as petalling I aim to create an artefact that accurately simulates different scenarios that would reflect real life, with the aim of incorporating it into the games industry and future works.


Low Level Platform Optimisation

Optimising a previously coded program and measuring the results:

At the start of the module I was given a framework that generates a number of cubes and spheres that collide with eachother. However, the program did not run particularly well to begin with. So, through the use of different optimisation techniques, it was my responisbilty to ensure the program runs faster and uses less resources whilst maintaining the original program mechanics; such as using octrees and using a memory pool.
